# stewieFinalDesign
•	基于 MEVN 技术栈（MongoDB、Express、Vue、Node.js） 开发旅行游记分享平台，采用 前后端分离架构，实现用户注册登录、游记发布、浏览、评论、点赞和收藏等核心功能。
•	使用 MongoDB 文档型数据库 设计用户、游记、评论、点赞等数据模型，支持非结构化数据存储，并通过 全文索引与正则匹配 实现游记关键词搜索。
•	实现 标签体系与兴趣推荐机制，根据用户兴趣标签匹配游记内容，并结合浏览量、点赞数等指标进行热门内容排序。
•	基于 JWT 实现用户身份认证，结合 RBAC 权限模型 开发后台管理系统，实现游记审核、评论管理与敏感词过滤等内容治理功能。
•	通过 Axios 实现前后端数据交互，使用 Quill 富文本编辑器 支持图文混排的游记创作，提高用户内容发布体验。
